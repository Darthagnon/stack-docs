# 国际化 (i18n)

Hugo 内置了对多语言站点的支持。您可以在 [官方文档](https://gohugo.io/content-management/multilingual/) 中找到更多信息。

翻译文件放置在 `i18n` 目录中。文件名是语言代码。例如，`en.yaml` 文件是英语的翻译文件。

为了使用某种语言，请在配置文件中将 `DefaultContentLanguage` 设置为语言代码。例如，如果您想使用英语，请将 `DefaultContentLanguage` 设置为 `en`。

目前，此主题支持以下语言：

* `ar`: 阿拉伯语
* `bn`: 孟加拉语
* `ca`: 加泰罗尼亚语
* `de`: 德语
* `el`: 希腊语
* `en`: 英语
* `es`: 西班牙语
* `fa`: 波斯语
* `fr`: 法语
* `hu`: 匈牙利语
* `id`: 印尼语
* `it`: 意大利语
* `ja`: 日语
* `ko`: 韩语
* `nl`: 荷兰语
* `pl`: 波兰语
* `pt-br`: 葡萄牙语
* `ru`: 俄语
* `th`: 泰语
* `tr`: 土耳其语
* `uk`: 乌克兰语
* `zh-cn`: 简体中文
* `zh-hk`: 繁体中文 (香港)
* `zh-tw`: 繁体中文 (台湾)

::: tip
欢迎通过 PR 支持更多语言 😉。
:::
