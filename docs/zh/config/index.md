# 介绍

Hugo 接受 `TOML`、`YAML` 和 `JSON` 作为配置格式。此主题目前使用 `YAML` 和 `TOML` 作为配置格式。

::: info
在可预见的将来，此主题将把所有配置迁移到 `TOML` 格式。
:::

如果您使用的是 `hugo-theme-stack-starter` 模板，您可以在 `config/_default/` 文件夹下找到配置文件，这些文件按主要章节分开，以便更容易找到您想要修改的配置。

可用配置的完整列表可以在此主题根目录下的 `config.yaml` 文件中找到。（如果不在那里，请发送 issue 告诉我。）

有很多工具可以将 `YAML` 转换为 `TOML`，反之亦然。您可以使用它们将配置文件转换为您喜欢的格式。
